FCC Image Search Abstraction Layer

To perfom an image search copy and past the following into the search bar: /api/image_search/query . Replace the word query with whatever you would like to search for.

Try these examples:

http://localhost:8888/api/image_search/dogs
http://localhost:8888/api/image_search/logs

Search terms can be followed by ?offset= and a number to view additional search results.

http://localhost:8888/api/image_search/frogs?offset=2

To view a list of recent searches paste /api/history into address bar.

http://localhost:8888/api/history
